# Лабораторная работа №4 #

Лабораторная работа по дисциплине "Микроконтроллеры".

Работа с таймерами в STM32. Приёмник.

## Как скачать проект?

### Способ первый: прямое скачивание архива.

Необходимо нажать кнопку "Clone or Download" на данной странице, далее нажать "Download ZIP". Архив с проектом скачается на ваш компьютер.

### Способ второй: с помощью git.

Необходимо установить себе на компьютер систему управления версиями git. Её можно скачать с [официального сайта](https://git-scm.com/downloads).

Затем в той папке, куда хотите положить проект, необходимо нажать правой кнопкой мыши и в меню выбрать Git Bash Here. В окне ввести следующую команду:

`git clone https://github.com/Alouettesu/lab4с.git` 

## Работа с проектом

В папке EWARM находится файл рабочего пространства среды IAR Embedded Workbench. Откройте его с помощью IAR, и проект готов к работе. Программа, содержащаяся в нём, готова к работе на отладочной плате.
